# Brave D Art Preview Store

واجهة متجر عربية بسيطة لعرض دفاتر أنيقة قبل الإطلاق الرسمي.

## الفكرة

هذا المشروع عبارة عن صفحة هبوط ومتجر عرض أولي لعلامة منزلية تبيع دفاتر أنيقة. الهدف منه:

- تقديم شكل بصري راقٍ يمكن مشاركته بسرعة مع مالك المتجر
- عرض مجموعة منتجات أولية مع أسعار وروابط طلب مباشرة
- تسهيل النشر المجاني على GitHub Pages

## الملفات

- `index.html` الواجهة الرئيسية
- `styles.css` التصميم والاستجابة للشاشات
- `app.js` بيانات المنتجات وتوليد بطاقات العرض

## المعاينة المحلية

بما أن المشروع ثابت، يكفي فتح `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub.
2. ارفع الملفات إلى الفرع `main`.
3. من إعدادات المستودع فعّل `GitHub Pages` من الفرع `main` والجذر `/`.
4. سيصبح المتجر متاحاً عبر رابط عام يمكن مشاركته.

## تخصيص سريع

- عدّل رابط واتساب داخل `index.html` و `app.js`
- حدّث نصوص المنتجات داخل `app.js`
- يمكن لاحقاً إضافة صفحة تفاصيل، سلة مشتريات، أو تكامل طلبات فعلي
